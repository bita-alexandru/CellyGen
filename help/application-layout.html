Application Layout

The application features 3 main panels:

  1. Input Panel
  2. Grid Panel
  3. GA Playground Panel

The Input Panel is formed by the following sub-panels:

  • States Panel
  • Neighbours Panel
  • Rules Panel

The Grid Panel is formed by the following sub-panels:

  • Grid Upper Tools Panel
  • An N x M grid you can manipulate using the upper and bottom toolbars
  • Grid Bottom Tools Panel

The GA Playground Panel is formed by the following sub-panels:

  • A panel representing the Parameters on which the algorithm should run
  • A panel designed to Run the G.A and observe the results
