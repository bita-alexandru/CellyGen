Defining States

Defining a state takes the following syntax form:

  • <name_of_the_state>;
    Examples:
    • STATE_A;
    • STATE_A; STATE B;
    • STATE_A;
      STATE B; STATE C;

You can define up to 255 states in this fashion as long as all of your states follow these rules:

  • the name is no longer than 64 characters
  • the name includes only characters of the English alphabet or underscores "_"
  • there are no duplicate states

The state FREE is already defined by default and each white cell on the grid is, by design, of this state.

Use ! to mark a comment so that everything that follows after gets ignored.
Example:
!this is a comment
STATE_A; !this is another comment
