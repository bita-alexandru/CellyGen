Grid Bottom Tools

This toolbar features the following main elements:

  • A panel for changing the delay of the simulation
  • A panel for manipulating the grid and the simulation
  • A panel for displaying the information about the simulation

The Delay Panel is formed of the following elements:

  1. Decrease button
    2. Decrease the delay of the simulation
  2. Increase button
    3. Increase the delay of the simulation

The Control Panel is formed of the following elements:

  1. Reset button
    2. Clear and reset the whole grid
  2. Go To Center button
    3. Go to the center of the grid (at the coordinates 0,0)
  3. Play/Pause button
    4. Play/Pause the whole simulation
  4. Next Generation button
    5. Play out the next generation of the current simulation
  5. Randomize panel
    6. Randomly populate the grid according to the factor set in the spin box

The Simulation Information Panel is a pair of labels displaying the current both the generation count and the population count at the current simulation state.
