Rules Panel

The Rules Panel is formed by the following elements:

  • A button for launching the Rules Editor
  • A search bar for searching into the list of rules
  • A list which displays all the available rules

The states list features a context menu that pops up when the right click button is triggered onto a rule.
The following options are available:

  • Go To
    Launch the Rules Editor and go to the line with the selected rule
  • Delete
    Delete the rule
