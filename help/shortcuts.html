Shortcuts

Shortcut Action
Alt+F4 Closes the application
Ctrl+O Import a Cellular Automaton configuration file
Ctrl+S Export a Cellular Automaton configuration file
Ctrl+Enter Launch the Grid Dimensions settings
Ctrl+1 Launch the States Editor
Ctrl+2 Launch the Rules Editor
Shift+1 Set the perspective to Default (reset the layout)
Shift+2 Set the perspective to Grid (layout focus on the grid)
F11 Switch to/out of Full Screen
F1 Launch the User Manual
Ctrl+Shift+1 Focus the States Search Bar
Ctrl+Shift+2 Focus the Rules Search Bar
Enter (on the search bar) Go to the next result
Shift+Enter (on the search bar) Go to the previous result
Ctrl++
Ctrl+ScrollIn (on the grid)		 Zoom into the grid
Ctrl+-
Ctrl+ScrollOut (on the grid)		 Zoom out of the grid
Ctrl+Z Undo the last change made on the grid
Ctrl+Shift+Z Redo the last change made on the grid
Ctrl+Q Switch into Drawing mode (enables drawing onto the grid)
Ctrl+W Switch into Picking mode (enables selecting states from the grid)
Ctrl+E Switch into Moving mode (enables panning around the grid)
Ctrl+LeftArrow
Shift+LeftClick		 Select previous state
Ctrl+RightArrow
Shift+RightClick		 Select next state
Alt+LeftClick (on the grid) Fill in the whole visible and non-occupied grid area with the select state
Ctrl+RightClick (on the grid) Erase the whole selected structure formed by cells of the clicked state
Ctrl+Alt+RightClick (on the grid) Erase the whole selected structure formed by cells of any state
Ctrl+, Decrease the simulation delay
Ctrl+. Increase the simulation delay
Ctrl+R Reset the grid
Ctrl+M Go to the center of the grid
Ctrl+Space Play/Pause the whole simulation
Ctrl+G Play out the next generation of the current simulation
Ctrl+P Randomly populate the grid
