States Panel

The States Panel is formed by the following elements:

  • A button for launching the States Editor
  • A search bar for searching into the list of states
  • A list which displays all the available states

The states list features a context menu that pops up when the right click button is triggered onto a state.
The following options are available:

  • Select
    Select the state
  • Go To
    Launch the States Editor and go to the line with the selected state
  • Change Color
    Change the color of the state
  • Erase
    Erase all the cells of that state
  • Delete
    Delete the state and the cells of that state
